## 🎯 리뉴얼 핵심 목표

### 현재 사이트 진단 (3줄 요약)
1. **사업영역은 다양하나(6대 사업)**, 메뉴가 50개로 분산되어 가치 전달이 어렵다
2. **CEO 개인 브랜드 vs 회사 브랜드** 정체성이 혼재되어 있다
3. **콘텐츠 빈 페이지가 많아** 신뢰도와 전문성을 떨어뜨린다

### 리뉴얼 방향
> "복잡한 사업을 단순하게, CEO와 회사 브랜드를 명확하게, 비어있는 곳은 채우거나 정리하기"

## 3. 디자인 방향 제안

### 톤앤매너
| 키워드 | 표현 |
|--------|------|
| **고급스러움** | 청담 럭셔리 라운지 컨셉 반영 |
| **문화·예술** | 클래식 + 모던 조화 |

| **글로벌** | 영문/한글 병기, 다국어 |

### 컬러 제안 (예시)
- **Primary**: 딥 네이비 / 와인 (럭셔리)
- **Accent**: 골드 / 샴페인 (클래식 우아)
- **Neutral**: 아이보리 / 차콜 그레이

### 핵심 디자인 요소
1. **풀스크린 비주얼**: 메인 히어로 영상·이미지
2. **카드형 사업 분류**: 6대 사업을 한 화면에 카드로
3. **타임라인**: 이서진 CEO 미디어 출연·수상 이력 비주얼화
4. **갤러리 그리드**: Pinterest 스타일 마소너리 레이아웃
5. **모바일 우선**: SNS 유입 → 모바일 70% 이상 가정

## 6. 기술 스택 제안

### 현재
- **플랫폼**: 아임웹 (Imweb)
- **장점**: 빠른 구축, 결제·게시판 등 기본 기능 내장
- **단점**: 디자인 자유도 한계, 고급 기능 추가 어려움

### 옵션 비교

#### 옵션 A: 아임웹 유지 + 디자인 리뉴얼
- **장점**: 운영 부담 적음, 비용 저렴
- **단점**: 디자인·UX 한계
- **적합 케이스**: 빠른 리뉴얼, 예산 제한

#### 옵션 B: 아임웹 → 워드프레스 또는 카페24
- **장점**: 디자인 자유도, 플러그인 풍부
- **단점**: 유지보수 인력 필요
- **적합 케이스**: 콘텐츠 마케팅 강화

#### 옵션 C: 풀 커스텀 (Next.js + Headless CMS)
- **장점**: 최고의 성능, 디자인, 확장성
- **단점**: 개발 비용·시간↑
- **적합 케이스**: 글로벌 브랜드로 도약, 의료관광 본격 진출

### 권장: 옵션 B 또는 옵션 A → C 단계적 마이그레이션

## ✅ 결론

**더컬쳐앤은 사업 모델은 매력적이지만, 그것이 홈페이지에 충분히 드러나지 않고 있습니다.**

리뉴얼의 핵심은:
1. **흩어진 50개 페이지를 5개 핵심 메뉴로 정리**
2. **빈 페이지(의료관광·VR·영문)는 채우거나 통폐합**
3. **CEO 브랜드 + 회사 브랜드의 균형 잡기**
4. **글로벌·다국어 지원으로 의료관광 사업 활성화**
5. **모바일 중심의 고급스러운 디자인 + 갤러리·영상 강조**

이를 통해 더컬쳐앤이 가진 "All in One" 가치를 시각적으로도 전달하는 것이 리뉴얼의 핵심 미션입니다.
